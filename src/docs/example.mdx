# 欢迎使用 JayWorks 插件

这是一个 MDX 文档示例，当前阶段展示：

- Frontmatter 解析
- 基础 Markdown 渲染
- 尚未激活的组件标签占位说明

## 功能介绍（当前可用）

- 支持 Markdown 语法
- 解析 frontmatter 元数据
- 代码块、列表、链接等常规结构

## 组件占位说明

计划中的交互组件（例如 `<SimpleButton />`、`<OcrPlayground />`）暂未在此阶段启用。

```mdx
// 未来可用示例（暂不可执行）
<SimpleButton label="点击我" action="notice" />
<OcrPlayground initialText="Hello" />
```

> 后续将通过安全的 mdxJsx → HAST 转换与 action 映射恢复组件能力。
