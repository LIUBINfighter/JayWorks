本页汇总原先分散在 `demo.mdx` 与 `example.mdx` 中的内容，聚焦最终用户最关心的：插件内置文档视图能做什么、你能在这里学到什么。

## 已实现能力

- 支持标准 Markdown 语法（标题、列表、引用、代码块等）。
- 解析 frontmatter 元数据（标题、描述、标签、更新时间等）。
- 支持受控的 MDX 组件（安全白名单）。
- 代码高亮与（未来的）Mermaid 图表渲染。

## 交互式组件示例

<SimpleButton label="点击我" />
<OcrPlayground initialText="Hello" />

> 目前仅支持字面量属性；表达式/对象/函数会被忽略（详见“开发者 → 渲染管线”）。

## 典型使用场景

| 场景 | 说明 |
| ---- | ---- |
| 快速阅读内置帮助 | 无需跳转外部网站，离线可用 |
| 查看交互示例 | 直接在文档里试用组件（按钮、Playground 等） |
| 跟随分步指引 | 未来可扩展为更详细的操作教程 |
