JayWorks 是一个面向 Obsidian 插件作者与最终使用者的“内置文档 + 交互演示”模板：

- 不再来回切换浏览器与编辑器：核心用法、示例、参考文档全部内嵌。
- 基于 Markdown / MDX：既可写纯文本，也可插入受控 React 组件。
- 结构清晰：区分“指南 / 参考 / 开发者”三类受众。

## 为什么内置文档？

> 80% 的常用帮助与入门信息无需跳转外部网站。

- 加快学习反馈：看到 → 点击 → 获得结果。
- 统一上下文：与插件 UI / 数据结构保持一致（无需重新解释）。
- 可离线：无网络时仍然可查阅。

## 文档结构

| 分组 | 目的 | 典型内容 |
| ---- | ---- | ---- |
| 指南（Guide） | 帮助最终用户上手 | 介绍、核心功能、图表示例 |
| 参考（Reference） | 查阅固定结构信息 | （占位）插件设置 |
| 开发者（Developer） | 插件模板扩展/维护 | 渲染管线、导航配置、Frontmatter 规范、样式指南、调试记录 |

## 我应该从哪里开始？

1. 浏览本页了解定位。
2. 打开“核心功能概览”试用交互式组件。
3. 需要图表？查看“交互式图表（Mermaid）”。
4. 想定制或扩展？前往“开发者”分组。
